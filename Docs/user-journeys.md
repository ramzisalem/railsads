🧠 User Journeys
This document defines the main flows through the product.

🎯 1. First-Time User Journey (CRITICAL)

Goal
Get user to their first successful creative as fast as possible

Flow

Step 1 — Entry
User lands on product

Step 2 — Website Import
User pastes website URL
System extracts:
- Brand
- Products
- Messaging

Step 3 — Product Selection
User selects products
(Default: all selected)

Step 4 — ICP Generation
System generates ICPs for selected products
User reviews and selects one

Step 5 — Creative Generation
User enters Creative Studio
System generates:
- Hooks
- Headlines
- Primary text

Step 6 — First Iteration
User modifies:
"Make it more emotional"

Step 7 — Selection
User selects best creative

Outcome
User gets their first usable ad

Key Moment
“This actually works” → retention trigger

🔁 2. Returning User Journey

Goal
Continue and improve existing creatives

Flow
Open thread
Review previous work
Iterate
Generate variations
Select best output
Export

Behavior
Less guidance needed
More iteration-focused

🧠 3. Exploration Journey

Goal
Discover new creative directions

Flow
Select product
Change ICP
Try new templates
Generate variations
Compare outputs

📊 4. Scaling Journey (Advanced Users)

Goal
Produce multiple creatives for testing

Flow
Generate multiple hooks
Create variations
Select best candidates
Export for ads platform

🧾 5. Empty State Journey

Goal
Guide user to first action

Pattern
Illustration
Clear message
Single CTA

⚠️ Critical UX Rules

Rule 1
Never leave user without next step

Rule 2
Always guide first-time users

Rule 3
Returning users should feel fast and efficient

Rule 4
First success must happen within minutes

🧾 Summary

User Journeys

1. First-time → guided, structured
2. Returning → fast, iterative
3. Exploration → flexible
4. Scaling → efficient

Goal:
Reduce friction from idea → usable ad

🧠 Final Insight
Users don’t come to use your product

They come to get ads that work
