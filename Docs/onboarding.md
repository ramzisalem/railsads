🧠 Onboarding System
Onboarding is the most critical UX moment.
Goal:
Get the user to their first successful ad in under 3 minutes

Core principle
Show value immediately, reduce thinking, guide every step

🎯 1. Onboarding Strategy

What we optimize for
Speed to value
Clarity
Momentum
Confidence

What we avoid
Long onboarding
Tutorials
Too many decisions
Empty screens

Golden rule
User should never ask: "what do I do next?"

🧩 2. Onboarding Flow (Main)

Step 1 — Entry

Screen
Input: Paste your website
CTA: "Import brand"

UX
Single input
No distractions
Strong value messaging

Microcopy
Paste your website and we’ll generate your brand, products, and messaging

Step 2 — Brand Import (Magic Moment)

System does:
Extract:
- Brand identity
- Products
- Messaging
- Images

UX
Show loading + progress:
"Analyzing your website..."

Important
👉 This is your first WOW moment

Step 3 — Product Selection

Screen
List of products
(All selected by default)

UX
User can:
- Keep all
- Deselect

Rule
No complex decisions here

Step 4 — ICP Generation

System generates:
3–5 ICPs per product

UI
Cards
Clear summaries
Select one

UX
Highlight recommended ICP

Step 5 — Enter Creative Studio

System auto-generates:
Hooks
Headlines
Primary texts

UX
No blank state
No waiting for user input

👉 This is the second WOW moment

Step 6 — First Interaction

Suggested prompts
Make it more emotional
Try a UGC style
Shorter version

UX
Clickable suggestions

Step 7 — Selection

User selects:
Best creative

UX
Clear "Use this" action

🧠 3. Onboarding Rules

Rule 1
Remove decisions early

Rule 2
Pre-fill everything possible

Rule 3
Always show output before asking input

Rule 4
Guide with suggestions

Rule 5
Make progress visible

⚠️ Common Mistakes

❌ Asking too much upfront
Brand name?
Audience?
Tone?
👉 NO

❌ Empty screens
"Start by creating..."
👉 NO

❌ Delayed value
User waits before seeing output

Onboarding Flow
