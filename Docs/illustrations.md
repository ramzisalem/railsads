🧠 Illustration System
Illustrations are not decorative.
They exist to:
Support the UI
Add warmth
Guide the user

Core principle
Illustrations are subtle, abstract, and secondary

🎯 1. Role of Illustrations

Used for
Empty states
Onboarding
Light background accents

Not used for
Core UI
Navigation
Main content

Intensity rule
80% → no illustrations
20% → minimal illustrations

🎨 2. Style Definition

Visual style
Abstract
Geometric
Soft
Organic shapes
Minimal detail

Mood
Calm
Modern
Neutral
Premium

Avoid
Characters
Cartoons
3D blobs
Playful styles
Overly colorful visuals

🎨 3. Color System
Use your design tokens:

Base
Beige → #F5F3EF
White → #FFFFFF

Secondary (structure)
Sage → #9FB5AE
Sage dark → #7F9992

Accent (very limited)
Orange → #FF6A00

Rules
Sage = main illustration color
Orange = small highlight only
Never dominate with orange

✏️ 4. Composition Rules

Structure
2–4 shapes max
No clutter
Lots of whitespace

Layout
Centered or slightly offset
Soft layering
Subtle depth

Shapes
Rounded rectangles
Circles
Blobs (controlled)
Lines (minimal)

🤖 5. AI Generation Guidelines

Base prompt template
Minimal abstract geometric illustration, soft beige background, sage green shapes, subtle orange accent, modern SaaS style, clean composition, no characters, no text, soft shadows, high-end design

Variations

Calm background illustration
Abstract geometric composition with soft sage shapes on warm beige background, minimal, clean, modern SaaS style, no text, no characters, subtle layering

Slightly dynamic (for onboarding)
Abstract flowing shapes with soft gradients in sage and beige, minimal orange accent, modern product illustration, calm and premium, no characters

Structured (for product-related states)
Minimal geometric layout with blocks and containers, sage tones, beige background, subtle orange highlight, clean UI-inspired illustration, no text

🧩 6. Use Cases

1. Empty state — Creative Studio
Prompt
Minimal abstract geometric illustration, soft beige background, layered sage shapes, subtle orange accent, calm and modern SaaS style, no characters

2. Empty state — Competitors
Prompt
Abstract composition with multiple soft geometric shapes representing structure and comparison, sage tones on beige background, minimal orange highlight

3. Empty state — Products
Prompt
Minimal structured layout with container-like shapes, soft sage palette, beige background, subtle depth, clean SaaS illustration style

4. Onboarding
Prompt
Abstract flowing shapes with layered composition, soft sage and beige tones, minimal orange accent, modern and premium product illustration, calm atmosphere

⚙️ 7. Output Specifications

Format
PNG or SVG
Transparent background (optional)

Size
1024x1024 (recommended)
or
square ratio (1:1)

Quality
Clean edges
No noise
No artifacts

🧠 8. Implementation Guidelines

Placement
Centered in empty states
Max width: 240–320px

Spacing
Margin-bottom → 24px

Opacity (optional)
80–90% for subtle effect

Background
Always on beige or neutral surface

⚠️ 9. Common Mistakes

❌ Too detailed
Feels busy and cheap

❌ Too colorful
Breaks calm identity

❌ Too large
Overpowers UI

❌ Too frequent
Loses impact

🧾 10. Summary

Illustration System

Style:
Abstract, geometric, minimal

Colors:
Beige + sage + subtle orange

Usage:
Only for empty states and onboarding

AI:
Generate using strict prompt templates

Goal:
Support the UI, not dominate it

🧠 Final Insight
Illustrations should feel like part of the system,
not an extra layer on top of it
