🧠 Jobs To Be Done
Jobs To Be Done define why users use the product, not how.

🎯 Core Job
Turn a product into high-performing ad creatives quickly and efficiently

🧩 Primary Jobs

1. Create ads fast
When I have a product,
I want to generate multiple ad creatives quickly,
so I can start testing without delay

2. Explore different angles
When I don’t know what will work,
I want to generate different angles and hooks,
so I can discover what resonates

3. Improve existing creatives
When I have a draft,
I want to iterate and refine it,
so I can improve performance without starting over

4. Understand my audience
When I select a product,
I want to understand the target audience,
so I can tailor messaging effectively

5. Reduce creative effort
When I create ads,
I want the system to guide me,
so I don’t need to think from scratch every time

🧠 Emotional Jobs

Feel confident in the output
Feel in control of the process
Avoid frustration and overwhelm

⚠️ Anti-Jobs (what users don’t want)

Write long prompts
Start from scratch every time
Lose previous work
Guess what works

🧾 Summary
Users are not trying to “use AI”

They are trying to:
- create ads faster
- explore ideas
- improve performance

The product must optimize for these outcomes
